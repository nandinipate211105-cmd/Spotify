# 🎵 Spotify Clone (Simplified)

A simplified version of Spotify built using **HTML, CSS, and JavaScript**. This project recreates the basic user interface and core music player functionality, providing a clean and responsive web experience.

## 🚀 Features

* 🎵 Play and pause songs
* ⏭️ Next and previous track controls
* 📀 Song list with selectable tracks
* 🎚️ Progress bar for playback
* 🔊 Interactive music player interface
* 📱 Responsive and modern design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Spotify-Clone/
│── index.html
|── Spotify.html
│── style.css
│── Spotify.css
│── Spotify.js
│── songs/
│── assets/
└── README.md
```

## 🎯 Purpose

This project was developed to practice front-end web development concepts, including responsive layouts, DOM manipulation, event handling, and JavaScript-based audio controls. It demonstrates how HTML, CSS, and JavaScript work together to create an interactive music player.

## 📌 Note

This is a simplified Spotify-inspired project created for educational and learning purposes. It is not affiliated with or intended to replace the official Spotify platform.
